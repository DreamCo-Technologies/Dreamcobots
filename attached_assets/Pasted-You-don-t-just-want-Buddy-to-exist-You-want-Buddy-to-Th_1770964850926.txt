3️⃣ AUTONOMY CONTROL — ENGINE LEVEL

Every bot receives:

A. Autonomy Mode
	•	Manual → User approves each step
	•	Assisted → Buddy suggests, user confirms
	•	Semi → Auto with checkpoints
	•	High → Full execution with logs

B. Aggression Mode

Controls:
	•	Max iterations
	•	Tool call frequency
	•	Retry logic
	•	Research depth

Example:
	•	Slow: 5 iterations, human checkpoint
	•	Medium: 20 iterations
	•	Aggressive: 100+ loops if needed

But always:
	•	Hard cap for cost protection
	•	Hard cap for runtime
