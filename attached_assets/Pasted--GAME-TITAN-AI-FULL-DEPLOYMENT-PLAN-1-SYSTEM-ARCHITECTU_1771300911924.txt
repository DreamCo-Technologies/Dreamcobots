2️⃣ MEMORY & RL LOOP

Autonomous learning cycle:
	1.	Observe game state → extract features
	2.	Compare current state to previous states
	3.	Predict optimal action → evaluate reward
	4.	Execute action → log outcome
	5.	Update RL model → adjust policy
	6.	Store strategy in long-term memory
	7.	Repeat for session; sync across sessions
	8.	Share learned patterns with DreamCo memory system

Meta-learning:
	•	Transfer knowledge between games
	•	Adapt strategy to player style
	•	Adjust difficulty and resource optimization dynamically
