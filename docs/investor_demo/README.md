# DreamCo Investor Demo Package

This package provides a self-contained investor presentation flow:

1. Live Revenue Simulator overview
2. Swarm visualization + consensus highlights
3. BuddyAI live prompt testing walkthrough
4. ROI snapshot and evolution metrics
5. Pitch flow and talking points

## Quick Launch

- Open `docs/investor_demo/index.html` in a browser, or
- Run `make investor-demo` from repository root.

## Demo Runtime Notes

- Deterministic simulated metrics are used for presentation reliability.
- Integrates with Control Tower Actions hub via the **Investor Demo** tab.
- Keeps governance/safety positioning visible across all sections.
