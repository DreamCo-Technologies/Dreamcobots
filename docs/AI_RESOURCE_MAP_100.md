## Extraction standard

For each resource, bots must produce:

- What it teaches DreamCo.
- Which bots need it.
- Which workflow it improves.
- What file, prompt, test, or dashboard component should change.
- Evidence that the idea works.
- Risks or limits.
- Next learning checkpoint.
