# Bot Contract Discovery System

Every DreamCo bot should continuously search for contracts and opportunity paths that match its division, capabilities, client type, and business model.

## What Bots Can Do By Default

- Search public and owner-approved sources.
- Summarize contract, grant, RFP, vendor, supplier, trucking route, and partnership opportunities.
- Score fit, risk, deadline, proof needed, eligibility, and value.
- Prepare response checklists and approval packets.
- Create weekly owner/client opportunity digests.

## What Requires Approval

Bots must ask before submitting bids, filing grant applications, registering vendor accounts, contacting buyers, contacting suppliers, contacting carriers, signing contracts, accepting terms, spending money, sharing client data, or making price commitments.

## Commands

```bash
npm run report:contract-discovery
npm run check:contract-discovery
```
