# AI Data Package Library

DreamCo can sell or license AI data packages only when the data is rights-cleared, source-backed, privacy-reviewed, quality-scored, and packaged for a clear buyer use case.

## Package Types

- Instruction tuning data.
- RAG knowledge bases.
- Evaluation benchmarks.
- Agent simulation datasets.
- Tool-calling datasets.
- Domain glossaries.
- Conversation safety datasets.
- Synthetic business cases.
- Vision labeling datasets.
- Consent-safe audio and voice datasets.
- Code agent tasks.
- Contract opportunity datasets.

## LangChain Readiness

The library is designed for LangChain-based loaders, chunkers, metadata normalization, retrieval datasets, structured output validation, and agent eval sets.

Required packages:

- `langchain`
- `@langchain/core`
- `zod`

## Rights Rule

Do not sell or license data unless DreamCo has the right to do so. Block secrets, unapproved private customer data, personal data without privacy review, copyrighted content without license, paywalled content without resale rights, and voice/face/likeness data without consent proof.
