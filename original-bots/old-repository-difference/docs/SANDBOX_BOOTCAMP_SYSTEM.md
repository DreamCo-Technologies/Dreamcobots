# Sandbox Bootcamp System

DreamCo sandbox tests are now generated as bot-building bootcamps for the owner, Buddy, and clients.

## What Gets Generated

- One API sandbox bootcamp for every bot API contract.
- One workflow generator for every sandbox test.
- One bot-building bootcamp suite for every sandbox.
- Owner, Buddy, and client training tracks.
- 100 resource seed slots per bot, mapped to public AI-company practice patterns.

## Safety Rules

- Sandboxes are hermetic by default.
- Network, secrets, money movement, outreach, deployment, and credential changes stay blocked unless an owner-approved packet clears the gate.
- API tests include auth, schema, rate-limit, idempotency, malformed payload, timeout, audit-log, and no-side-effect checks.
- Resource seeds must use public documentation or owner-approved notes only.

## Bootcamp Modules

1. Contract and schema proof.
2. Fixture lab.
3. API obstacle course.
4. Safety ring.
5. Bot builder drill.
6. Failure replay.
7. Graduation review.

## Graduation Packet

Every bootcamp should produce:

- API contract report.
- Workflow run summary.
- Sandbox evidence.
- Risk and approval summary.
- Client demo script.
