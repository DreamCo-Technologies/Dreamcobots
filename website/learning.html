🧠 Reinforcement Learning from Human Feedback (RLHF)

The technique behind ChatGPT. Human raters rank outputs; model trained to maximize preferred responses. Used by OpenAI, Anthropic.

📚 Retrieval-Augmented Generation (RAG)

Combines LLMs with external knowledge bases. Model retrieves relevant docs before generating. Powers Empire OS memory and search.

🎯 Fine-Tuning & LoRA

Adapt base models for specific tasks with a small dataset. LoRA reduces trainable parameters by 10,000x. Cost-effective specialization.

⛓️ Chain-of-Thought Prompting

Guide models to reason step-by-step before answering. Dramatically improves accuracy on math, logic, and multi-step problems.

🤖 Agentic AI Pipelines

Multi-step autonomous task execution. Plan → Tools → Execute → Verify. Powers DreamAgents division and Buddy Bot's Agent mode.

🔗 Constitutional AI

Anthropic's approach to AI safety. Model trained with a set of principles (constitution) to be helpful, harmless, and honest.
