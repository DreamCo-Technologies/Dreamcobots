# Buddy Commerce and Publishing OS

Make Buddy the download-easy, task-manager-first commerce and publishing operator for domains, app builds, web publishing, app-store readiness, device installs, and safe autonomous money research.

- Readiness score: 85

## Commerce Lanes

### Domain Selling and Management

- Safe actions: search domain ideas, compare public registrar pricing, draft domain shortlist, prepare DNS plan, prepare renewal reminders
- Approval required: buy_domain, transfer_domain, change_dns, sell_domain, connect_payment_method

### Self-Publishing Apps

- Safe actions: build release checklist, create screenshots checklist, draft privacy and support notes, prepare staged rollout plan, prepare rollback checklist
- Approval required: submit_store_listing, accept_store_terms, publish_public_release, connect_paid_account

### Easy Client Downloads

- Safe actions: prepare web link, prepare PWA install instructions, prepare desktop package checklist, prepare mobile store packet, track device support
- Approval required: ship_installer, submit_mobile_app, collect_client_device_data, send_external_download_link

### Task Manager Chatbot

- Safe actions: capture goals, split tasks, assign bots, track status, summarize blockers, draft next actions, store useful memory only
- Approval required: create_external_task, invite_user, send_notification, change_client_account

### Autonomous Money Research

- Safe actions: search public opportunities, compare customer pains, draft offers, score revenue potential, prepare outreach drafts, simulate pricing
- Approval required: contact_customer, publish_offer, run_ads, charge_payment, bid_contract, scrape_non_public_data

## Download Targets

- Web App: Open Buddy from any browser with no install. (ready_for_static_preview)
