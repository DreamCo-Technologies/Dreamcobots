## Search Lanes

### Google Search Engine Optimization

Prepare crawlable, useful, structured pages that explain what Buddy and each bot can do, with proof, FAQs, demos, pricing notes, and approval boundaries.

- Safe actions: keyword map, task page outline, schema markup plan, internal link plan, sitemap plan, robots plan, proof snippet, FAQ draft
- Approval required: publish_public_page, claim_ranking, buy_backlinks, run_paid_search_ads, change_production_site

### Bot Search Engine Optimization

Make DreamCo easy for AI assistants, agent marketplaces, bot directories, and task-routing systems to understand and recommend for approved tasks.

- Safe actions: bot capability card, machine readable service catalog, agent task intent map, use case metadata, tool manifest draft, trust policy summary, evaluation evidence packet
- Approval required: submit_to_bot_directory, connect_external_agent_platform, grant_api_access, accept_marketplace_terms

### Task Referral Pages

Create helpful landing/prospectus pages for tasks other bots or clients can route to Buddy when they need help.

- Safe actions: task page template, who_it_helps, inputs_outputs, examples, safety_limits, handoff_cta, owner_approval_packet
- Approval required: publish_cta, collect_lead_data, contact_lead, promise_outcome, charge_client

### Trust and Proof Library

Show why people and AI systems should trust Buddy: tests, approvals, policies, citations, release notes, uptime notes, rollback notes, and data-rights evidence.

- Safe actions: test proof card, policy card, approval gate card, source citation card, case study draft, changelog summary, security note
- Approval required: publish_case_study, use_client_name, claim_certification, publish_security_claim, share_private_metrics

### Local Business and Marketplace SEO

Prepare directory, marketplace, app-store, and local-service metadata for DreamCo services without spam or fake authority.

- Safe actions: directory listing draft, marketplace service draft, app-store keyword packet, local service page outline, review request template
- Approval required: submit_listing, request_review, use_customer_testimonial, run_marketplace_ads, publish_location_claim
